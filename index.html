python-googlevoice by justquick

Python Interface for Google Voice

Google Voice for Python Allows you to place calls, send sms, download voicemail, and check the vaious folders of your Google Voice Accounts. You can use the Python API or command line script to schedule calls, check for new recieved calls/sms, or even sync your recorded voicemails/calls
